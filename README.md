## **✨ 주요 기능**  
- **캐릭터 상호작용**  
  - 캐릭터가 공에 맞으면 공이 튕겨 나가는 물리 효과 구현.  
  - 배구 규칙을 활용한 점수 시스템.  
- **점수 시스템**  
  - 공이 바닥에 닿을 경우 마지막으로 공을 맞춘 캐릭터에게 점수 부여.  
  - 먼저 **11점**을 획득한 플레이어가 승리.  

---

## **🕹 프로그램 사용법**  
1. **방향키**로 캐릭터를 조작합니다.  
2. 공을 캐릭터에 맞춰 상대방의 코트에 튕겨 넣으세요.  
3. **11점**을 먼저 획득하면 승리합니다!
